当前我使用页面生成工具

自动页面生成工具是最简单的页面制作方式，它能帮你排版出简易又漂亮的页面。

作为作者，你可以使用Github喜爱的标注方式，选择一个被设计师精工的模板，放出你的文章。

页面生成后你可以看看下面的分支： 

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

若是 GitHub Mac 轻松同步即出现新分支

设计师模板

我们制作了一些漂亮模板供你使用。从那里挑选一下吧。很容易改变这些。在发布之后你也可以从新选择另一个。只要你的标记格式不变。

手动强迫状？

如果不喜欢自动生成，push gh-pages 到你自己的repository 手动创建页面，不止支持常规的HTML， Github还可用 Jekyll 一个简单的类博客静态页面生成器，由我们的 Tom Preston-Werner 写成。 Jekyll makes it easy to create site-wide headers and footers without having to copy them across every page. It also offers intelligent blog support and other advanced templating features.
